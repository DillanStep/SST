# Contributing (Dashboard)

Thanks for helping improve the SST Dashboard.

## Setup

- Node 18+
- `cd apps/web`
- `npm install`
- `npm run dev`

## Code style

- Prefer TypeScript types over `any`
- Keep API calls in `src/services/`
- Run `npm run lint` before opening a PR
