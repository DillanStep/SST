# Requirements

## Server

- DayZ Server installed and runnable
- Write access to the DayZ profile directory (SST writes to `$profile:SST/`)

## Node API

- Node.js 18+
- Permissions to read/write the configured SST paths (inventories, events, api folder)

## Web dashboard

- Node.js + npm (for local dev) OR Docker (for container deployment)

## Optional (recommended)

- SSD storage for the DayZ profile folder
- A reverse proxy (Nginx/Caddy) for production hosting
